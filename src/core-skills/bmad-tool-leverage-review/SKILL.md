# Tool Leverage Review

Capability Improvement Toolkit skill for deciding whether existing BMAD skills,
repo scripts, MCPs, browser affordances, or automation should be used before
adding new work.

This skill is planning guidance only and has no runtime authority.

## Execution

1. List candidate tools that can provide evidence, speed, repeatability, or
   safety.
2. Decide use / skip / enhance for each candidate, with one-sentence rationale.
3. For OpenAI developer-product work, require OpenAI developer docs MCP as the
   preferred source for current OpenAI product facts.
4. Prefer existing tools when they cover the task with clear evidence.
5. Propose tool enhancement only when repeated work or safety risk justifies it.
