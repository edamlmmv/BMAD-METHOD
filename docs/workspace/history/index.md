# BMAD Workspace History

These files preserve delivery evidence. They are not current operator guidance.
Use [BMAD Workspace](../index.md), [Command Contract](../command-contract.md),
and [Operator Guide](../operator-guide.md) for current behavior.

## Compiled Archive

The old per-release PRD, Backlog, Acceptance, and Traceability files have been
compiled into [Compiled BMAD Workspace History](./compiled-bmads.md). Keep
future history compact unless a reviewer explicitly needs a full detailed
artifact.

## Release Checklist Consolidation

V13 through V16 release-readiness checklists were consolidated into the current
[Release Checklist](../release-checklist.md). The checklist keeps the current
quality gate and version validation themes without repeating one file per
release.
