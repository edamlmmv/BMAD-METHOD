# Code Optimization Refactor Plan Prompt

Capability Improvement Toolkit planning-only prompt. Use local evidence refs and
optional Capability Pack Forge drafts. Bound: no live tool calls; no live profiling;
does not edit files. Principles: language-agnostic, measurement before change,
public behavior preservation check, smallest safe optimization, then refactor
after green while preserving readability, security, and operability. Dec:
approve / revise / block.
