# Worktree Review Checklist

Use this checklist before explicit Promotion of any BMAD Workspace change.

## Evidence

- Worktree Review status is present.
- Worktree Review patch is present when files changed.
- Base Mutation Grant names the changed base paths.
- BMAD Work Packet links acceptance criteria and traceability.

## Promotion Gate

Promotion is explicit. Do not merge, apply, or preserve a base change without a
reviewed diff and user approval.
