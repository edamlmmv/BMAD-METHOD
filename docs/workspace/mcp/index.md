# Workspace MCP Planning

These planning notes describe MCP-related advisory evidence surfaces for BMAD
Workspace. They are documentation and routing guidance only: they do not grant
Workspace authority, change verifier input, configure MCP servers, install
connectors, prove runtime availability, or authorize writes.

## Boundary

Use these docs when an operator or BMAD skill needs to decide how MCP evidence
should be described, requested, or recorded. Keep sealed verifier evidence in
self-contained Capability Request JSON and evidence templates; live MCP host
state, local secrets, connector sessions, and target runtime behavior remain
outside Workspace verifier authority.
